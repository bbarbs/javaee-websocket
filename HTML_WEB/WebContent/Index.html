HTML5 Preliminary Study


This is a sample demo application using Html5 with Java EE.

Some of the features presented here is Restful WebService using Jersey for JAX-RS implementation,WebSocket, Jquery, Angularjs.

Demo



Restful WebService

List of api.

api description
api/equip/list This will retrieve the list of equipment.
api/equip/{id} This use the @PathParam to retrieve equipment by specific id/pk.
api/equip/get/equipment/{name} This use the @PathParam to retrieve equipment by specific name.
api/equip/get/search?equipName={equipName} This use the @QueryParam to retrieve equipment by their name.
api/equip/get/{activity}/{operator} This uses two @PathParam for retrieving data.

Here is some of the URI that can be used to access data in web service.
